DICE PUZZLE GAME: - (by Rishabh Saxena)

1. This project is a procedurally generated dice puzzle game developed using Python and Pygame.
2. The player controls a dice that rolls across an isometric tile-based map. The objective is to reach the goal tile ensuring that the required face of the dice is on top.

Features: -
1. Procedurally generated levels
2. Dice orientation mechanics
3. Multiple level types (Normal, Broken tiles, locked tiles)
4. Timer-based score system and gameplay

Controls: -
Arrow Keys/WASD = Move the dice

How to win: - 
Reach the end tile as fast as possible, with the correct face on the top of dice
Broken tiles break after stepped on them once
Locked tiles require a number on the top of the dice to unlock
(Hint : Try to cover all the tiles while reaching the end tile)

Extra Features: -
Solution path is generated alongside the level, ensuring each level can be completed.

Technologies used: -
Pygame (Python)
Pyinstaller (py to exe file)
